# IBM Full Stack Software Developer Capstone Project - Dealership Application

This is the final capstone project for the IBM Full Stack Software Developer Professional Certificate. It implements a nationwide car dealership website where users can view dealership branches, filter by state, view details of each dealer, view customer reviews with sentiment analysis, and submit new reviews.

## Architecture
- **Backend**: Django (Python) for user authentication, car make/model database, and proxy API services.
- **Microservices**: Node.js and Express with MongoDB for managing dealers and reviews, and a Python Sentiment Analyzer.
- **Frontend**: React.js with Bootstrap styling.
- **CI/CD**: GitHub Actions workflow.
